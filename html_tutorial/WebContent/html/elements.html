HTML Elements(요소)

목록

1. html 요소

HTML 엘리먼트는 시작 태그에서 종료 태그까지 모두이다 :
  1. <p>This is a paragraph</p>
  2. <a href="default.htm">This is a link</a>
  3. <br>
(시작 태그는 여는 태그, 종료 태그는 닫는 태그라고도 한다)

2. 중첩된 html 요소

대부분의 HTML 요소는 중첩 될 수있다. (다른 HTML 요소를 포함 할 수있다)
  1. <!DOCTYPE html>
  2. <html>
  3. <body>
  4.     <p>This is my first paragraph.</p>
  5. </body>
  6. </html>

3. 종료태그를 생략하지 않도록 주의하자

브라우즈나 태그에 종류에 따라 종료태그를 생략하여도 올바르게 표시될수도 있으나 종료태그를 생략하지 않도록 주의 한다.
  1. <p>This is a paragraph
  2. <p>This is a paragraph

4. 비어있는 html요소

내용부분이 비어있는 <br>같은 태그의 경우 닫는 태그없이 사용가능하지만, <XHTML>에서는 <br/>과 같이 표현해야 한다.

html에서 대/소문자

html에서는 특별히 대/소문자를 구분하지 않지만 <XHTML>에서는 대/소문자를 구분하니 예제는 소문자 태그를 사용한다.
